## Интеграция модулей из папки `V2/` (15 минут)

1. **Добавить файлы в таргет:** в Xcode перетащить папку `TamaDuckAlpha/V2`
   в проект (Create groups, галочка на watch-таргете).

2. **Capability:** Signing & Capabilities → `+ Capability` → **HealthKit**
   (для watch-таргета).

3. **Info.plist** (watch-таргет), новый ключ:
   - `NSHealthShareUsageDescription` =
     `TAMA радуется твоим шагам и кольцам активности и растёт вместе с тобой.`

4. **Авторизация при старте** — в `TamaDuckAlphaApp` (однократно, при появлении):
   ```swift
   .task {
       await HealthKitService.shared.requestAuthorization()
       await NotificationPlanner.requestAuthorization()
       await engine.syncActivity()
   }
   ```

5. **Обновление при активации сцены** — в существующий `onChange(of: scenePhase)`:
   ```swift
   if phase == .active {
       Task {
           await engine.syncActivity()
           await NotificationPlanner.syncChainNotifications(state: engine.pet)
       }
   }
   ```

6. **Пуши цепочек** — после каждого ответа/пропуска/переноса квеста
   (проще всего одним местом — в конце `answerQuest`, `skipQuest`
   и `finishQuest`):
   ```swift
   Task { await NotificationPlanner.syncChainNotifications(state: pet) }
   ```

7. **Alpha tools:** в MoreMenu полезно добавить кнопку
   `ActivityReactor.shared.resetForTesting()` — сброс дневных вех.

## Как проверить на часах (demo-режим)

1. Запусти, разреши Health и уведомления.
2. Пройдись по комнате 100 шагов → открой приложение: вехи 2000+ сработают,
   если шаги за день уже есть (снимок берётся за весь день, не с запуска).
3. Ответь на квест «красивая уточка» смело → сверни приложение →
   через 25–35 секунд придёт пуш «💌 Кажется, вам письмо…».
4. Закрой кольца (или дождись вечера) → праздник + 15 XP.

## Ограничения текущей реализации (честно)

- Снимок HealthKit обновляется при открытии приложения/активации сцены;
  фоновые observer-запросы и обновление complication — задача M5.
- Пуш дневного пинга ставится на календарное время и не знает,
  открывал ли ты приложение позже — возможен пинг после сыгранных квестов
  (безвредно, лечится в M2 доработкой).
- Память об активности — TODO: нужно подключить к текущему API
  `MemoryEngine.remember` (см. комментарий в `PetEngine+Activity.swift`).
